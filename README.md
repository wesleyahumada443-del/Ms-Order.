# ms-order-service
Este proyecto esta dearrollado en Java, consta del uso de diversas tecnologias como Spring Boot,Postgre Db, Kafka y Docker.
Con la finalidad de hacer uso practico de las mismas y que un usuario pueda estar al tanto de que sucede al momento de hacer una compra en linea, con los diversos escenarios que podrian pasar.
Solo tiene la parte de Back End, es decir, todo lo que el usuario no visualiza de manera directa.
Esta abierto para posibles mejoras o manipulaciones de distintos colaboradores.
